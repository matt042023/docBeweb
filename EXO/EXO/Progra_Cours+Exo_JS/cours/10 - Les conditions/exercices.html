Pour répondre à ces exercices, tapez votre code dans le fichier exercices.js.
N'oubliez pas d'ouvrir la console développeur de Chrome pour voir vos erreurs JavaScript.

Consulter la doc MDN sur les conditions

Objectifs

  • Comprendre le fonctionnement des conditions et des mot-clés if et else.
  • Utiliser les conditions pour résoudre des problèmes simples.
  • Détecter les bogues liés aux mauvaises pratiques et savoir les corriger.

Exercice 0

La société SensationsGaranties demande à Enzo de rédiger un programme pour afficher si un client peut monter dans un manège ou pas.

L'attraction sur laquelle il travaille autorise les clients à monter seulement s'ils ont 18 ans ou plus et s'ils font strictement moins de 120 kg.

Enzo a déclaré les variables, mais il a du mal avec la condition, aidez-le à finir son code !

Exercice 1

Un nouveau manège réservé aux enfants vient d'être réalisé, il faut écrire le code pour gérer son accès.

Évidemment c'est Enzo qui s'y colle. La règle du manège est que l'enfant doit faire strictement moins de 50 kg ou avoir 12 ans (ou moins) pour y accéder.

Enzo n'a pas eu le temps de rédiger la condition et de mettre les instructions d'affichage dans les bons blocs, finissez son code !

Exercice 2

Des frustrations se font sentir au manège pour adultes ! Les clients râlent et posent tous la même question : "Pourquoi je ne peux pas monter dans ce manège ?!".

En effet les messages actuels n'indiquent pas au client la raison pour laquelle il ne peut pas monter dans le manège ! La société SensationsGaranties demande à Enzo de corriger ça.

Rappel des règles : si les clients ont 18 ans (ou plus) et font strictement moins de 120 kg, ils peuvent monter.

Le nouveau programme sera un peu plus utile pour le client :

  • Si le client peut monter, on affiche Bienvenue dans le manège !
  • Si le client ne peut pas monter, on affiche soit :
    • S'il est trop lourd : Vous ne pouvez pas monter : les sièges ne supporteraient pas votre poids.
    • S'il est trop jeune : Vous ne pouvez pas monter : ce manège est réservé aux adultes.
    • S'il est trop lourd et trop jeune : Vous devriez vite arrêter de manger des frites !

Enzo est un peu perdu, il n'a pas trop idée de comment faire, aidez-le !

Cet exercice risque d'être difficile pour vous donc prenez votre temps !
