Les chaînes de caractères

Pour répondre à ces exercices, tapez votre code dans le fichier exercices.js.
N'oubliez pas d'ouvrir la console développeur de Chrome pour voir vos erreurs JavaScript.

Consulter la doc MDN sur les chaînes de caractères

Objectifs

  • Utiliser l'instruction console.log() pour afficher des chaînes de caractères.
  • Échapper des caractères avec \.
  • Concaténer plusieurs chaînes de caractères avec l'opérateur +.
  • Utiliser les littéraux de gabarit `` pour construire des chaînes de caractères.

Chassez le bogue ! 0

Enzo vient juste de se faire embaucher dans la société MagiciensDuCode en tant que développeur JavaScript junior.

Aujourd'hui, son patron Hugo vient le voir car il prépare une grosse campagne marketing. Il a besoin d'envoyer une lettre personnalisée à chaque client qui commence comme ça :

Bonjour <prénom du client>, aujourd'hui nous faisons une promo "la fidélité paye" !

Enzo doit écrire une chaîne de caractère qui permet d'écrire cette phrase pour chaque client. Pour l'instant il essaye de faire fonctionner son code avec le prénom "Bob".

Malheureusement son code ne fonctionne pas, pouvez-vous corriger ses erreurs ?

Chassez le bogue ! 1

Hugo vient voir Enzo pour lui demander de formater un peu le début de sa lettre car on ne plaisante pas avec le marketing chez MagiciensDuCode.

Il aimerait qu'elle ressemble plutôt à ça :

Aujourd'hui nous faisons une promo "la fidélité paye" !
